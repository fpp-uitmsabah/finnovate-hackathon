# FIN-novate Hackathon 2025 - Demo Applications

This directory contains example demo applications that showcase possible solutions to the FIN-novate Hackathon challenges. These demos serve as inspiration and reference for teams building their own solutions.

## üìÅ Demo Applications

### 1. Risk Confidence Builder (`fear-of-loss.html`)
**Challenge #1: The "Fear of Loss" Barrier**

An interactive calculator that helps students overcome fear of investing by showing:
- Real-time comparison of savings accounts vs. diversified investments
- Compound growth calculations over customizable time periods
- Inflation impact on purchasing power
- Multiple scenario profiles (Fresh Graduate, Mid-Career, Windfall)
- Visual charts demonstrating growth differences

**Features:**
- Interactive sliders for amount and time period
- Real-time calculations
- Educational insights about compound growth
- Mobile-responsive design

---

### 2. Smart Growth Simulator (`get-rich-quick.html`)
**Challenge #2: "Get Rich Quick" Mindset**

A simulation tool comparing risky "get rich quick" schemes with steady wealth building:
- 10-year timeline showing realistic outcomes
- Side-by-side comparison of two strategies
- Red flags to identify scams and schemes
- Principles of sustainable wealth building
- Visual timeline with year-by-year progression

**Features:**
- Interactive strategy selector
- Animated timeline visualization
- Warning boxes for scam detection
- Educational principles grid
- Success/failure outcome scenarios

---

### 3. Risk Profile Analyzer (`risk-profile.html`)
**Challenge #3: Misunderstanding Risk Appetite**

A comprehensive quiz-based tool for discovering investment personality:
- 6-question assessment covering multiple risk dimensions
- 3 distinct investor personas (Guardian, Balanced Planner, Growth Seeker)
- Personalized recommendations based on profile
- Detailed characteristics breakdown
- Investment strategy guidance

**Features:**
- Progress tracking through questions
- Previous/Next navigation
- Detailed persona results with recommendations
- Risk tolerance metrics
- Mobile-optimized quiz interface

---

### 4. Investment Vehicle Explorer (`stock-vs-unittrust.html`)
**Challenge #8: Stock vs. Unit Trust Confusion**

An educational platform clarifying differences between stocks and unit trusts:
- Side-by-side feature comparison
- 3 tabbed sections: Overview, Scenarios, Quiz
- Real-world scenario analysis
- Interactive knowledge quiz
- Pros and cons breakdown

**Features:**
- Tabbed interface for organized content
- Scenario-based learning
- Interactive quiz with feedback
- Comparison tables
- When-to-choose guidelines

---

### 5. Smart Invest-Buddy (`first-step-paralysis.html`)
**Challenge #7: "First Step" Paralysis**

A gamified mobile app prototype making investment approachable:
- Personalized onboarding with risk assessment
- Interactive learning modules
- Sandbox portfolio simulation
- AI chatbot support
- Leaderboard and gamification elements

**Features:**
- Multi-screen mobile app simulation
- XP and achievement system
- Progress tracking
- Educational content modules
- Practice trading without risk

## üé® Design Principles

All demos follow these principles:
- **Mobile-First**: Responsive design works on all devices
- **Interactive**: Users learn by doing, not just reading
- **Visual**: Charts, animations, and visual feedback enhance understanding
- **Educational**: Clear explanations accompany all features
- **Accessible**: Clean, readable interfaces with good color contrast
